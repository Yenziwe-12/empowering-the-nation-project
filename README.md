## Project Overview

Empowering the Nation is a web and mobile application developed to support a Johannesburg-based skills training organisation. The platform provides information on a variety of domestic and gardening upskilling programmes and allows users to request quotations based on selected courses.

The system was developed as part of a Work Integrated Learning (WIL) programme, with an emphasis on **practical application**, **user-centred design**, and **professional real-world development practices**.

---

## Key Features

- Professional branding and interface aligned to the organisation’s purpose
- Seamless User Experience across **mobile app** (React) and **website** (HTML, CSS, JavaScript)
- Course summaries and detailed pages for both *six-month* and *six-week* training programmes
- Interactive **Quote Calculator** applying bundled **discount rates** and **VAT**
- Contact page including organisation details, map location and enquiry functionality
- Clear navigation menus and responsive layout for accessibility and ease of use
